# Transbank Opencart 3.x Onepay Plugin

## Descripción

Este plugin de Opencart 3.x implementa el [SDK PHP de Onepay](https://github.com/TransbankDevelopers/transbank-sdk-php) en modalidad checkout. 

## Dependencias

* transbank/transbank-sdk
* setasign/fpdf
